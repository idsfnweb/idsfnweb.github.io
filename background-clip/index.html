How To create a translucent border or padding without background color

Applying a translucent dotted border to a solid background.

Part2. How to achieve a translucent border ?

let's start with a red box

then let's add a solid blue border

after that let's make it a bit translucent, to do that we must convert HEX to HSLA or RGBA, that way we can control the opacity.

#34495e is equal to rgba(52, 73, 94, 1)

now replace the old HEX value within your border parameters, and choose the suitable opacity, something like this

Now we need to fix that border vs background clach

By default the background-color paints

  • The Content Box
  • The Padding Box
  • & The Border Box

All at once, DAMN!

Below is the CSS Box Model, so you could see what I'm talking about.

Also, you can inspect using the developer tools what we've just designed, to see clearly the box model in action

All we need to do is to change that behavior, we can specify where the painting must stop by using background-clip.

Here's all the properties that background-clip can take
background-clip: border-box|padding-box|content-box|initial|inherit;

Okay, back to our code, let's add background-clip: padding-box;

Cool, but what if I added a little bit of padding to the box?
The box will grow and that extra space will be painted in red :D, if this is not what you desire, go ahead and change background-clip.

background-clip: content-box;

this is the end result

Part3. Apply the knowledge.

let's start with a simple F letter

<h2>F</h2>

I'm kinda trying to create a social facebook button, so probably you should wrap F in <a> tag.

Next, I want to add a light blue background and color F with dark blue

as you see, F, ain't in the center, let's center it Horizontally and vertically.

Cool, I guess it's time to add a tiny hover effect to our button.
On hover the background and the letter will switch colors, also the mouse cursor will change to pointer
as you see below.

it'll be much better if we slow down the transition,
also adding ease-in-out will make it so smooth.

Nice, let's add border to the left and a little of bit of padding to separate the left border with the actual background.

border-left: 0.1em solid #3b5999;  
padding-left: 0.1em;

You'll notice that padding-left didn't actually create any separation. WHY ?
I guess you already know, but if you didn't follow part1, the background paints the whole CSS Box Model by default, including the padding box and the border box.

That's why you saw an increase in size but you didn't see any separation between the border and the background.
And as I said before to change this behavior we need to use background-clip.

After adding background-clip: content-box; that's what we have so far, as a result

Note. I changed padding-left to a higher value to make it obvious.

Inside h2:hover let's add this

padding-left: 0;  
transition: background 0.3s ease-in-out,  
            padding 0.2s linear;

setting padding to O and adding a bit of transition will make the button move toward the border in a slick motion.

but wait, where's the border ?

the border now is in the same background color, so let's give it a different color.

´border-left: 0.2em solid #ecf0f1;´

By far this is what we have
