Part1. How to achieve multiple borders in CSS3

In the upcoming sections, we'll learn how to create more than one border using multiple techniques.

1.1 Multiple borders via box-shadow

There's a fourth parameter that box-shadow accept, it's called spread radius.
If we combined spread radius with zero radius and zero blur, like you see below

background-color: #c0392b;  
box-shadow: 0 0 0 1em #ecf0f1;

We get a solid border. Something like this

See the Pen MbmjMM by IDSFN (@IDSFN) on CodePen.

If we want to add another border, we need to stack another line to box-shadow with a comma separate.

box-shadow: 0 0 0 1em red,   /* border 1 */  
            0 0 0 2em blue,  /* border 2 */
            0 0 0 3em black, /* border 3 */
            0 0 0 4em pink;  /* border 4 */

In our previous example, border1 is our top layer, border2 is underneath border1, etc ...
All box-shadows are overlaid one on top of the other, always add the SUM of previous layer sizes + the desired new border size.

for example, if I wanted border3 to be 3em in size: this is how you should do the math :D

(border1 size + border2 size) + border3 desired size

that means

(1em + 2em) + 3em = 6em.

What about the top, right, bottom or left?

You won't need radius for that, vertical shadow or horizontal shadow is enough.
The example below, will make it clear.

box-shadow: 0 -1em 0 red,    /* top border */  
            1em 0 0 blue,    /* right border */
            0 1em 0 black,   /* bottom border */
            -1em 0 0 yellow; /* left border */

here you can see it live

See the Pen ZBKBoO by IDSFN (@IDSFN) on CodePen.

notes :

  • Use the keyword inset, if you wanna capture mouse events on these fake borders.
  • Shadows don't affect layout, not like borders. The box-sizing property won't count them unless you put them inside the box, by using inset.
1.2 Multiple borders via outline

Bad news first, outline can only produce one border, but if we combine outline & border we'll have a total of 2.

here's an example :

border: 1em solid black;   /* Border 1 */  
outline: 1em solid yellow; /* Border 2 */

The downside of using outline is sharp corners, sadly outlines do not follow rounding through border-radius.

1.3 Multiple borders via background clip

I dedicated a whole episode for background clip. I'll move fast, but if you have any difficulties you can read the previous episode here.

By controlling the painting area using background-clip, you can fake it to make it look like you have a border.

here's an example :

See the Pen bBWLbx by IDSFN (@IDSFN) on CodePen.

As you see, i implemented only right borders, and I've used a mix of box-shadow, border and background-clip.
The first white border was a result of using background-clip, to break it down. Here's how!

background-color: #f1c40f;  
background-clip: content-box;  
/*
  Setting background clip to content box will make
  Background-color only paint the content box,
  padding & border box will stay untouched
*/

padding-right: 1em; /* This is our fake border */  
border-right: 1em solid #e67e22; /* This is border2 */

Part2. Apply it in a real life example

There's no need to make things complicated, all you need to do is play with shades.

Something like this:

.card2 {
  box-shadow: 0.08em 0.03em 0 #C2C2C8, /*Strong color*/
              0.2em 0.1em 0 #EDEDF4,   /*less stronger*/
              0.3em 0.2em 0 #EEEEF5,   /*less less*/
              0.3em 0.3em 0 #F0F0F6;   /*less less less*/
  }

It's trial and error, the more borders you add, the more depth you'll have. It'll take a little bit of experiment, but you'll get there.
